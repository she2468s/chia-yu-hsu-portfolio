Cloud Website

Type  Website Design 網頁設計
Cloud Overview
Cloud Website 讓使用者可以在網頁上管理放在 Cloud Storage 裡的檔案以及 Cloud 訂閱方案。
在 Overview 頁面，使用者可以一覽使用者資訊、雲端儲存空間用量、訂閱的方案以及儲存在雲端內的檔案。使用者可以在此通往使用者資訊編輯的頁面，也可以進行訂閱方案的管理，並且瀏覽最新上傳的檔案。
Cloud Files
Files 頁面則是雲端檔案管理系統，使用者可以上傳檔案或資料夾後，進行分享、複製、移動、預覽等動作。

Subscription Management
